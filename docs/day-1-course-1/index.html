심화 ─ 필터링 및 형태학적 변환

3. 이미지 필터링 (Thresholding & Histogram)

임곗값을 기반으로 이미지를 이진화하는 기법과, 명암비를 높이는 히스토그램 스트레칭을 실습합니다. 영수증(receipt) 이미지와 lena 이미지를 활용합니다.

📖 이론 설명

📌 주요 함수 레퍼런스

함수 역할 핵심 파라미터 (기능, 범위/형식, 예시) 공식 문서
cv2.threshold() 전역 임곗값 이진화
  • src: 그레이스케일 입력 이미지
  • thresh: 임곗값. Otsu 사용 시 0으로 지정 가능
  • maxval: 임곗값을 넘었을 때 할당할 픽셀 값 (보통 255)
  • type: cv2.THRESH_BINARY. Otsu 적용 시 + cv2.THRESH_OTSU 추가
  • 예시: cv2.threshold(img, 0, 255, cv2.THRESH_BINARY | cv2.THRESH_OTSU)
 Link
cv2.adaptiveThreshold() 적응형 임곗값 이진화
  • src: 그레이스케일 이미지
  • maxValue: 최대값 (255)
  • adaptiveMethod: 임곗값 계산 방법. cv2.ADAPTIVE_THRESH_GAUSSIAN_C 등
  • blockSize: 주변 영역 크기 (홀수, 예: 11)
  • C: 계산된 평균에서 뺄 상수 값 (예: 2)
  • 예시: cv2.adaptiveThreshold(img, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C, cv2.THRESH_BINARY, 11, 2)
 Link
cv2.normalize() 배열 값 정규화
  • src: 입력 배열
  • dst: 출력 배열 (보통 None 전달 후 리턴받음)
  • alpha, beta: 정규화 범위의 하한과 상한 (예: 0, 255)
  • norm_type: 정규화 방식. 최소-최대 스트레칭은 cv2.NORM_MINMAX
  • 예시: cv2.normalize(img, None, 0, 255, cv2.NORM_MINMAX)
 Link

4. 형태학적 변환 (Morphological Transform)

이진화된 이미지에서 노이즈를 제거하거나 객체의 형태를 보완하기 위해 침식(Erosion)과 팽창(Dilation)을 사용합니다. 실습용으로 앞서 생성한 영수증 Otsu 이진화 이미지를 활용해봅니다.

📖 이론 설명

💡 참고: 분수령(Watershed) 알고리즘이란?
이미지를 지형(높낮이)으로 간주하여 물을 채운다고 가정했을 때, 물이 만나는 경계선(분수령)을 찾아 객체를 분리하는 고급 분할 기법입니다. 동전처럼 겹쳐있는 객체를 분리할 때 유용하지만, 본 기초 과정에서는 개념만 이해하고 넘어갑니다.

📌 주요 함수 레퍼런스

함수 역할 핵심 파라미터 (기능, 범위/형식, 예시) 공식 문서
cv2.erode() 침식 연산
  • src: 이진화된 입력 이미지
  • kernel: 구조 요소 (연산을 수행할 픽셀 매트릭스, 예: 3x3 1배열)
  • iterations: 반복 횟수
  • 예시: cv2.erode(img, kernel, iterations=1)
 Link
cv2.dilate() 팽창 연산
  • src: 이진화된 입력 이미지
  • kernel: 구조 요소
  • 예시: cv2.dilate(img, kernel, iterations=1)
 Link
cv2.morphologyEx() 고급 형태학 연산
  • src: 입력 이미지
  • op: 연산 종류 (cv2.MORPH_OPEN(열기), cv2.MORPH_CLOSE(닫기) 등)
  • kernel: 구조 요소
  • 예시: cv2.morphologyEx(img, cv2.MORPH_OPEN, kernel)
 Link

2

cv2.dilate() — 밝은 객체를 확장시켜 끊어진 부분을 연결합니다.

고급 ─ 기하 알고리즘 기초

5. 카메라 기하학과 변환 (Perspective & solvePnP)

📚 기하 알고리즘 핵심 이론

  • 핀홀 카메라 모델 (Pinhole Camera): 빛이 작은 구멍을 통과해 상이 맺히는 원리로, 3D 공간의 점이 2D 이미지 평면에 투영되는 구조를 설명합니다.
  • 소실점과 소실선 (Vanishing Point/Line): 현실에서 평행한 두 선이 이미지 상에서 만나는 지점을 소실점이라 합니다. 이를 통해 지형이나 카메라의 높이, 초점 거리(Focal length) 추정이 가능합니다.
  • 투영 변환 (Perspective Transform): 3차원 공간의 객체를 다른 시점의 2차원 평면으로 투영시키는 변환입니다. 원근감을 조절하거나, 삐뚤어진 문서를 평면으로 펴는 작업(Homography)에 필수적으로 사용됩니다.
  • SolvePnP: 이미 알고 있는 3D 모델(예: 크기를 아는 A4 용지)의 좌표와 2D 이미지 상의 대응점 좌표를 바탕으로, 카메라의 위치와 자세(회전/병진)를 계산하는 알고리즘입니다.

📖 이론 설명
