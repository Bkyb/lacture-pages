1. 포인트 클라우드 기초 (참고)

이어지는 실습들을 위해 깊이 이미지(Depth Image)로부터 포인트 클라우드를 생성하는 기초 원리를 다룹니다. 카메라 내부 파라미터(Intrinsic)를 이용해 2D 픽셀 좌표와 깊이 값을 3D 공간 좌표로 변환합니다. 본 차시에서는 아래의 함수가 미리 정의되어 있다고 가정하고 실습을 진행합니다.

# 깊이 이미지에서 포인트 클라우드를 생성하는 기본 함수 (참고용)
def create_point_cloud_from_depth(depth_img, camera_matrix):
    h, w = depth_img.shape
    fx, fy = camera_matrix[0, 0], camera_matrix[1, 1]
    cx, cy = camera_matrix[0, 2], camera_matrix[1, 2]
    
    # 픽셀 좌표 그리드 생성
    u, v = np.meshgrid(np.arange(w), np.arange(h))
    
    # Z값(깊이)이 0보다 큰 유효한 픽셀만 추출
    valid = depth_img > 0
    z = depth_img[valid]
    
    # 핀홀 카메라 모델 역투영 수식: X = (u - cx) * Z / fx
    x = (u[valid] - cx) * z / fx
    y = (v[valid] - cy) * z / fy
    
    # (N, 3) 형태의 포인트 클라우드 배열 반환
    points = np.vstack((x, y, z)).T
    return points

2. solvePnP를 활용한 자세 추정

이전 차시에서 학습한 solvePnP 기법을 활용하여 60mm x 60mm x 100mm 크기의 나무 블록의 자세를 추정합니다. 육각형 형태의 실루엣에서 꼭짓점을 찾아내어 3D 공간 좌표와 매핑하는 것이 핵심입니다.

📖 이론 설명 (이전 차시 복습)

solvePnP는 3D 모델의 실제 좌표와 대응하는 2D 이미지 픽셀 좌표 간의 관계를 통해 카메라 기준 객체의 회전(rvec)과 병진(tvec) 벡터를 구합니다. 이번 실습에서는 나무 블록의 외곽 형태(육각형)에서 기하학적 특징(소실점, 교차점)을 추론하여 대응점을 찾습니다.

cv2.solvePnP() 객체 6D 자세 추정 objectPoints, imagePoints, cameraMatrix, distCoeffs

5. 로봇-카메라 좌표계 이해 (Hand-Eye Calibration)

로봇의 움직임과 카메라 관측 결과 사이의 관계를 수학적으로 정의하여, 카메라 좌표계와 로봇 좌표계를 일치시키는 캘리브레이션 과정을 실습합니다.

📖 이론 설명

📚 캘리브레이션 수식(AX=XB)의 의미

  • 카메라가 로봇 팔 끝에 부착된 Eye-in-Hand 시스템의 경우, 고정된 환경(체스보드 등)을 다각도에서 촬영합니다.
  • 로봇의 이동(A)과 카메라가 바라본 타겟의 상대적 이동(B) 간의 루프를 닫아줌으로써, 센서와 로봇 플랜지 사이의 고정된 변환 행렬(X)을 수치 최적화로 구합니다.
