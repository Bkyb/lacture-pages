1. YOLO 모델 기초

YOLO(You Only Look Once)는 단일 신경망을 통해 이미지 전체를 한 번에 평가하여 객체의 경계 상자(Bounding Box)와 클래스 확률을 동시에 예측하는 One-Stage Detector입니다. 최신 버전인 YOLO11을 사용하여 기초 추론을 진행해 봅니다.

📖 이론 설명

📚 YOLO 아키텍처 구조

  • Backbone (백본): 입력 이미지로부터 다양한 크기의 유의미한 피처(Feature Map)를 추출하는 역할을 합니다. (예: CSPDarknet 등)
  • Neck (넥): 백본에서 추출된 피처맵들을 결합하고 혼합하여(FPN, PANet 등) 다양한 크기의 객체를 잘 탐지할 수 있도록 정보 해상도를 높입니다.
  • Head (헤드): 최종적으로 객체의 위치(BBox)와 클래스(Class)를 예측하고 출력합니다.

📌 주요 함수 레퍼런스

함수 역할 핵심 파라미터
YOLO() 사전학습된 YOLO 모델 로드 model_path ('yolo11n.pt' 등)
model.predict() 이미지/비디오 추론 수행 source, save, conf

2. Ultralytics 프레임워크 수정 및 응용 (심화)

이 세션에서는 프레임워크를 단순 블랙박스로 사용하지 않고, 내부 소스코드를 직접 수정하여 나만의 커스텀 아키텍처를 구성하는 방법을 학습합니다. 백본(Backbone) 교체, 손실 함수(Loss) 변경, 평가지표(Metric) 튜닝 등을 수행할 수 있습니다. 코딩 실습보다 아키텍처 튜닝 워크플로를 익히는 데 목적이 있습니다.

📖 기본적인 방법 (Basic Method)

새로운 블록이나 손실 함수를 모델에 추가하기 위해서는 프레임워크가 이를 인식할 수 있도록 일련의 등록 절차를 거쳐야 합니다.
  1. 모듈 정의: ultralytics/nn/modules 폴더 내 (예: block.py)에 PyTorch 기반의 클래스를 생성합니다.
  2. 초기화 등록: ultralytics/nn/modules/__init__.py 파일의 __all__ = (...) 튜플에 방금 만든 클래스 이름을 추가합니다.
  3. 파서 등록: ultralytics/nn/tasks.py 상단의 from ultralytics.nn.modules import ... 에 클래스를 추가합니다. (필요시 parse_model 함수 내부 로직도 수정합니다.)
  4. 설정 적용: 모델 구조를 정의하는 커스텀 .yaml 파일에서 새로 만든 모듈 이름과 인자(args)를 사용합니다.

✨ 사례 1: 새로운 백본(Backbone) 모듈 추가하기

1) 모듈 정의 (ultralytics/nn/modules/block.py)

timm 라이브러리를 활용하여 다양한 네트워크를 가져오는 TimmBackbone과 피처맵 분리를 위한 SelectFeature를 추가합니다.

class TimmBackbone(nn.Module):
    def __init__(self, model_name='efficientnetv2_rw_t', pretrained=True, out_indices=(2, 3, 4)):
        super().__init__()
        self.model = timm.create_model(
            model_name, features_only=True, out_indices=out_indices, pretrained=pretrained
        )

    def forward(self, x):
        feats = self.model(x)  # list of tensors
        return feats 

class SelectFeature(nn.Module):
    def __init__(self, index, c_out):
        super().__init__()
        self.index = index

    def forward(self, x):
        return x[self.index]

✨ 사례 2: 새로운 손실함수(Loss Function) 적용하기

ultralytics/utils/loss.py 수정 (Focal Loss 적용)

YOLO 모델의 클래스 분류 손실(Class loss)을 기본 BCE에서 Focal Loss로 교체하는 예시입니다.

import torch.nn.functional as F

# 1. 사용할 Loss Function 정의 (loss.py 파일 내)
class FocalLoss(nn.Module):
    def __init__(self):
        super().__init__()

    @staticmethod
    def forward(pred, label, gamma=1.5, alpha=0.25):
        loss = F.binary_cross_entropy_with_logits(pred, label, reduction="none")
        pred_prob = pred.sigmoid()
        p_t = label * pred_prob + (1 - label) * (1 - pred_prob)
        modulating_factor = (1.0 - p_t) ** gamma
        loss *= modulating_factor
        if alpha > 0:
            alpha_factor = label * alpha + (1 - label) * (1 - alpha)
            loss *= alpha_factor
        return loss.mean(1).sum()

3. SAM 2 & SAM 3 모델 사용해보기

📖 이론 설명

SAM (Segment Anything Model)은 Meta Research에서 발표한 혁신적인 Vision Foundation Model입니다. 단일 모델 하나로 제로샷(Zero-shot) 기반의 강력한 이미지 객체 분할 능력을 자랑합니다.

  • 프롬프트 엔지니어링: 점(Point)을 찍거나 박스(Box)를 그리는 등의 단순한 프롬프트만으로도 사용자가 원하는 픽셀 영역을 즉시 분할(Masking)합니다.
  • SAM 2: 이미지뿐만 아니라 비디오(Video) 프레임 간의 객체 추적 능력이 더해졌으며, 구조가 더욱 빠르고 가벼워졌습니다.
  • SAM 3 (최신/확장): 보다 고도화된 텍스트 기반 프롬프팅 등을 지원하여, 라벨링 인력과 비용을 획기적으로 절감할 수 있는 자동화 파이프라인 구축의 핵심이 됩니다.

from google.colab import output
from IPython.display import HTML, display
import base64
import cv2
import numpy as np

clicked_points = []

def show_interactive_image(image_array):
    _, buffer = cv2.imencode('.png', cv2.cvtColor(image_array, cv2.COLOR_RGB2BGR))
    img_base64 = base64.b64encode(buffer).decode('utf-8')
    h, w = image_array.shape[:2]

    html_code = f'''
    <div style="position: relative; display: inline-block;">
        <canvas id="canvas" width="{w}" height="{h}"
            style="border:1px solid black; cursor:crosshair; background-image: url('data:image/png;base64,{img_base64}');">
        </canvas>
        <p>클릭된 좌표: <span id="coords">없음</span></p>
        <button onclick="google.colab.kernel.invokeFunction('notebook.ClearPoints', [], {{}})">초기화</button>
    </div>
    <script>
        var canvas = document.getElementById('canvas');
        var ctx = canvas.getContext('2d');
        canvas.onclick = function(e) {{
            var rect = canvas.getBoundingClientRect();
            var x = Math.round(e.clientX - rect.left);
            var y = Math.round(e.clientY - rect.top);
            
            ctx.fillStyle = "red"; ctx.beginPath(); ctx.arc(x, y, 4, 0, Math.PI * 2); ctx.fill();
            document.getElementById('coords').innerHTML = "X: " + x + ", Y: " + y;
            google.colab.kernel.invokeFunction('notebook.AddPoint', [x, y], {{}});
        }};
    </script>
    '''
    display(HTML(html_code))

def add_point(x, y):
    clicked_points.append((x, y))
    print(f"좌표 추가됨: ({x}, {y})")

def clear_points():
    global clicked_points
    clicked_points = []
    print("좌표가 초기화되었습니다.")

output.register_callback('notebook.AddPoint', add_point)
output.register_callback('notebook.ClearPoints', clear_points)
